# VideoPlayerManager 广告机项目
一款视频播放器，支持边播边缓存、预缓存. 列表循环播放

增加开机自启动，崩溃重启

-对MVP进行了封装 

-基于protobuf的数据结构

-增加定时任务

-增加Banner轮播图

-本地文件夹下视频循环播放
